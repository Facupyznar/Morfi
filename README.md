
- Para instalar las librerias solo hay que correr un comando desde el directorio Morfi pip install -r requirements.txt

- Si se agrega o se elimina alguna libreria hay que actualizar el archivo requirements.txt, con una linea de codigo. pip freeze > requirements.txt

- Se hace commit y push del archivo y despues los otros tienen que hacer git pull y correr el codigo para descargar las nuevas librerias. pip install -r requirements.txt
